# LifePilot AI

AI-powered life planning mobile app built with React Native and Expo.

## Tech Stack

- **Frontend:** React Native, Expo
- **Language:** TypeScript
- **Backend:** Firebase
- **AI:** Integrated AI planning features

## Getting Started

```bash
cp .env.example .env
yarn install
npx expo start
```
